# Secure File Encryption System

A Python-based cybersecurity tool that encrypts and decrypts files using AES encryption with a simple GUI interface.

## 📌 Project Overview

The Secure File Encryption System is a cybersecurity project developed using Python.  
It allows users to securely encrypt and decrypt files to protect sensitive data from unauthorized access.

The system uses **AES symmetric encryption** through the Python **cryptography** library.  
A simple graphical user interface (GUI) built with **Tkinter** allows users to select files from their system and perform encryption or decryption operations easily.

This project demonstrates fundamental **applied cryptography concepts** such as secure key generation, encryption, and data protection.


## 🚀 Features

- Secure file encryption using AES encryption
- File decryption using the same secret key
- Simple GUI interface for file selection
- Automatic encryption key generation
- Protection of sensitive files from unauthorized access


## 🛠 Technologies Used

- Python
- Cryptography Library
- Tkinter (GUI)
- Git
- GitHub


## ⚙️ How It Works

1. Generate a secret encryption key.
2. Select a file from the system.
3. Encrypt the file using AES encryption.
4. Decrypt the encrypted file using the same key.
